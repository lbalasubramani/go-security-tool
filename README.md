# go-security-tool

Enterprise Security Tooling Platform

Inspired by Spotify & Netflix internal security dashboards.

## Features
- Trivy + Kube-bench scanning
- Falco runtime security
- Grafana dashboard integration
- Threat modeling (STRIDE + DFD)
- OWASP ZAP + vulnerability management

## Quick Start
```bash
go run cmd/security-tool/main.go
```
